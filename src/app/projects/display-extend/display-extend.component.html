DISPLAY EXTEND
🖥️🔌📱
Display Extend is a software solution that extends the screen real-estate of Windows computers by using an Android mobile device as an additional monitor. By establishing a USB connection between the Android device and the Windows PC, Display Extend creates a virtual display that streams content to the mobile device, providing users with an extended screen space.
Display Extend running on Android Tablet
Display Extend comprises two components: a Python server designed to run on a Windows PC and a client application developed for Android mobile devices. When the Android device is connected to the Windows computer via USB, the server and client automatically detect each other and establish a connection to enable the creation of the virtual display.
Virtual Display 🖥️
Display Extend's server component generates a fully virtual display that is transmitted over the USB connection to the client device. This virtual display is entirely software-based and eliminates the need for additional hardware, such as a dummy plug. The virtual display seamlessly integrates with the Android device, providing users with an extended screen for improved multitasking and productivity.
Automatic Size and Adjustment 📏
During the startup process, the server and client communicate to determine the optimal screen size for the connected mobile device. This intelligent sizing negotiation ensures that the virtual display perfectly suits the Android device. Furthermore, Display Extend dynamically adjusts the stream whenever the display's position is modified by the user within the "Display Settings" Windows menu.
h.264 Video Encoding 🎥
To facilitate real-time streaming, Display Extend captures the video stream from the Windows PC and encodes it as an H264 (AVC) data stream using ffmpeg. The encoded stream is then transmitted over a websockets connection, accessible to the client device using Android Debug Bridge (ADB) and reverse port forwarding.
