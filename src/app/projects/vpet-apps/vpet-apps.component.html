Alpha Hub Website
Alpha is a project which builds technology to interface with Bandai's Virtual Pet toy range - specifically, the Digimon line originally released in 1997. As a spin-off of Bandai's original Tamagotchi line, the Digimon variant of Virtual Pet toys were released in an attempt to appeal to a young male audience.
However, in the two-decades since their initial release, this line of toys have gathered a worldwide cult-following.
Bandai Digimon 20th Anniversary Virtual Pet Toy

The Alpha project has a series of apps which enable users to connect their Digimon VPet (Virtual Pet) devices to a computer or android phone via USB, and battle them on the internet. This is achieved via a custom microcontroller-based USB adapter to translate the signals bi-directionally between the app running on the computer/phone and the digimon vpet device.
Alpha Serial
Alpha Serial App
a barebones app which allows a user to simply read the data from the VPet, and write data back to it. It displays it in a serial console interface. It lacks the data processing required for online device connections, but is a useful utility for reading and writing data directly.
Alpha Terminal
Alpha Terminal App
Alpha Terminal extends the capability of Alpha Terminal. It provides the user with a clean user interface, and handles all of the data processing under-the-hood, which is essential for letting users connect their devices over the internet.
Additionally, it encrypts the data and includes tampering detection to prevent users from cheating by artificially improving their odds of successful battles over the internet.
Alpha Link
A work-in-progess project, which further extends Alpha Terminal by combining the VPet interfacing technology with social networking features, to help users find each other and connect their devices online.
